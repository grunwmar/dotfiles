# dotfiles

**prerequisities**
`zsh`, `curl` / `wget`, `git`, `neovim`


Install by `curl`
```sh
curl -fsSL https://raw.githubusercontent.com/grunwmar/dotfiles/refs/heads/main/WEBINST | zsh -s -- -i
```

Installl by `wget`
```sh
wget https://raw.githubusercontent.com/grunwmar/dotfiles/refs/heads/main/WEBINST -O - | zsh -s -- -i
```
